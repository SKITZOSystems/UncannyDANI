# v1.0.0 - Stable Narrative System Lock

UncannyDANI v1.0.0 is the first complete, bounded release state of the system.

## Purpose
UncannyDANI is a single-screen, session-scoped, browser-based narrative terminal that simulates adaptive conversational drift, boot sequencing, and memory illusion mechanics within a controlled, non-persistent environment.

## Core Behavior
- Single-screen terminal interface
- Boot sequence before interaction
- Session-scoped memory illusion only
- Drift-based response variation
- Controlled reset and reboot cycle
- Deterministic build behavior

## Hard Boundaries
- No real user profiling or behavioral tracking across sessions
- No backend persistence or external storage
- No API integrations or external data sources
- No multi-user identity systems
- No AI training or model adaptation
- No multi-screen application expansion
- No authentication systems
- No real-world inference or psychological modeling claims

## Allowed Changes
- UI refinement
- Tone and drift tuning
- Boot pacing adjustments
- Readability improvements
- Performance optimizations

## Requires Version Bump
- New persistence systems
- New screens or routes
- New engine layers
- External integrations
- Identity expansion systems

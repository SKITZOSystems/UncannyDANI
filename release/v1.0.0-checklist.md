# v1.0.0 Acceptance Checklist

UncannyDANI is considered v1.0.0 complete when all items below remain true.

## Required
- The app runs as a single-screen terminal experience.
- The boot sequence always precedes interaction.
- Session continuity stays scoped to the current browser session.
- `/reset`, `reset`, `/reboot`, `reboot`, and `/recalibrate` restore the boot flow without reloading the page.
- Drift never breaks the UI loop or build output.
- No new core system is required to continue development.
- Builds remain deterministic and deployable to Cloudflare Pages.

## Verification
- `npm run build` passes.
- The preview server serves a valid page.
- Reset returns the session to a clean narrative start.
